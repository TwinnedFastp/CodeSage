### 认证模块（backend/services/auth_service.py + endpoints/auth.py）
- 注册：RFC 邮箱校验(email-validator) + 唯一性 + 密码复杂度(>=8位大小写数字特殊) + bcrypt + 24h验证链接
- 登录：仅已验证账号；5 次错误锁 15 分钟；记录 IP(加密)；JWT access 7d + refresh 30d
- 登出：access 进 Redis 黑名单 + 吊销该用户全部 refresh
- 刷新：refresh 白名单校验 + 旋转令牌（旧 refresh 失效）
- 邮件服务：SMTP 配置则异步发送(aiosmtplib)，否则开发模式打印链接到日志

### 会话存储（conversation_service.py + endpoints/conversations.py）
- 5 类数据 CRUD：原始聊天记录/会话摘要/用户偏好(JSONB)/重要事实(加密)/任务状态
- 全部按 user_id 隔离，事务+异常回滚
- user_facts.fact_value 用 Fernet 加密落库，API 返回明文

### 数据库
- 7 张表，合理索引（user+created 复合索引、UUID 唯一索引、外键级联）
- init_db.py 建表（启用 pgcrypto 扩展供 gen_random_uuid）

- FastAPI 依赖注入抛 AuthError 默认 500，加全局 exception_handler 统一转 JSON
